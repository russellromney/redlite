## COMMANDZ

### STRING COMMANDZ

```lolcode
BTW SET - PUTZ VALUE IN KEY
REDLITE'Z DB CALLIN SET WIF "key" AN "value"

BTW GET - GETZ VALUE FROM KEY
I HAS A VAL ITZ REDLITE'Z DB CALLIN GET WIF "key"
VAL R NOOB  BTW if key not exist, VAL iz NOOB

BTW SETEX - SET WIF EXPIRY (TIEMOUT KITTEH)
REDLITE'Z DB CALLIN SETEX WIF "key" AN 60 AN "value"

BTW DEL - MAEK IT GO AWAY
REDLITE'Z DB CALLIN DEL WIF "key"

BTW INCR - MOAR NUMBAH
I HAS A MOAR ITZ REDLITE'Z DB CALLIN INCR WIF "counter"

BTW DECR - LESS NUMBAH
I HAS A LESS ITZ REDLITE'Z DB CALLIN DECR WIF "counter"
```

### HASH COMMANDZ (BUCKIT)

```lolcode
BTW HSET - PUT IN BUCKIT
REDLITE'Z DB CALLIN HSET WIF "buckit" AN "field" AN "value"

BTW HGET - GET FROM BUCKIT
I HAS A VAL ITZ REDLITE'Z DB CALLIN HGET WIF "buckit" AN "field"

BTW HGETALL - GET ALL TEH THINGZ
I HAS A STUFFZ ITZ REDLITE'Z DB CALLIN HGETALL WIF "buckit"
```

### LIST COMMANDZ (CONGA LIEN)

```lolcode
BTW LPUSH - ADD 2 FRONT OF LIEN
REDLITE'Z DB CALLIN LPUSH WIF "lien" AN "kitteh"

BTW RPUSH - ADD 2 BACK OF LIEN
REDLITE'Z DB CALLIN RPUSH WIF "lien" AN "puppeh"

BTW LPOP - TAEK FROM FRONT
I HAS A FIRST ITZ REDLITE'Z DB CALLIN LPOP WIF "lien"

BTW LRANGE - SEE ALL IN LIEN
I HAS A ALL ITZ REDLITE'Z DB CALLIN LRANGE WIF "lien" AN 0 AN -1
```
